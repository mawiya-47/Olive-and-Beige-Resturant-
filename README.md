## 🍽️ About

**Olive & Beige** is a restaurant web app built with a modern React + TypeScript stack and powered by the **Google Gemini API**. It was bootstrapped in Google AI Studio and combines a fast Vite frontend with a lightweight Express backend to deliver a polished, animated dining experience — from browsing the menu to interacting with AI-assisted features.

The name itself sets the tone: *olive* and *beige* — an earthy, elegant palette meant to evoke a warm, upscale restaurant atmosphere.

---

## ✨ Features

- 🎨 **Elegant, warm-toned UI** styled with Tailwind CSS 4
- ⚡ **Blazing-fast dev experience** courtesy of Vite 6
- 🤖 **Gemini-powered AI integration** via `@google/genai`
- 🎬 **Smooth animations** using [Motion](https://motion.dev)
- 🧩 **Type-safe codebase** end-to-end with TypeScript
- 🖥️ **Custom Express server** for serving and API routing
- 🪶 **Clean icon system** with `lucide-react`

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) installed
- A [Gemini API key](https://ai.google.dev/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/mawiya-47/Olive-and-Beige-Resturant-.git
cd Olive-and-Beige-Resturant-

# 2. Install dependencies
npm install
```

### Configuration

Create a `.env.local` file in the project root and add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run locally

```bash
npm run dev
```

The app will start in development mode — open the local URL shown in your terminal to view it.
